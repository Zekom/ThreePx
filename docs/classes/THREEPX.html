THREEPX Class

Defined in: src\threepx.js:39

ThreePx synchronises the three.js frustrum with that of the CSS.
See working examples in the examples folder.

Constructor

THREEPX

(
  • wrapper
  • [options]
)

Defined in src\threepx.js:39

Parameters:

  • wrapper HTMLElement

    The THREEPX.renderer.domElement will be added at position 0 of the wrapper.

  • [options] Object optional

    The default options are:

    • configurable: false
      Enable this option to override if you want override the API.
    • autoInitializeThreejs: true
      Enable or disable the creation of the scene, the renderer and the camera on the new ThreePx instance.
    • autoResize: true
      Enable or disable the bind between the wrapper size and the THREEPX.renderer.domElement.
    • debug: false
      Enable or disable some three.js elements added to the scene to debug.

Example:

  var wrapperElement = document.getElementById('wrapper'),
      threepx = new THREEPX(wrapperElement),
      mesh = new THREE.Mesh(
          // define your mesh
      )
  ;

  threepx.scene.add(mesh);

  var lastTimestamp = new Date();

  threepx.renderFunction  = function (timestamp, width, height) {

      var delta = timestamp - lastTimestamp,
          angle = 0.02 * delta * 2 * Math.PI / 1000
      ;

      mesh.rotation.x += angle;
      mesh.rotation.y += angle;

      mesh.position.set(100 + width/2, 100 - height/2, 0);

      lastTimestamp = timestamp;

      return true;
  }
